# UPLOADER
Uploader is an dummy project to upload files and retrieve images to
- Local Directory
- MongoDB GridFs
- AWS S3

## Technology Used
- Jquery 
- Node
- Mongodb
## How to Start the project
Fill the constants in *constantServer.js*
Run command *npm start*
